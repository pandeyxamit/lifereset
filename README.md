# 66 Day Life Reset

A self-contained, offline-first **PWA** that turns a 66-day self-improvement
challenge into an RPG: daily **quests**, **XP / levels**, spaced-repetition
**flashcards** (Valmiki Ramayan deck included), **stats & achievements**,
workout **timers**, a **hydration** counter, **backup/restore**, and local
**reminders**. Tracking is time-locked to start **17 June 2026**.

All data stays on your device (`localStorage`). No accounts, no servers.

## Run it

It's a static site — just serve the folder over HTTPS (required for the service
worker / installability). Hosting is handled by **tinyhost**; any static host
works too.

Local preview:

```bash
python3 -m http.server 8080   # then open http://localhost:8080
```

## Install on iOS (full-screen, offline)

1. Open the hosted URL in **Safari**.
2. **Share → Add to Home Screen**.
3. Launch from the new icon — it runs full-screen and works offline.

## Features

- **Today** — streak banner, Physical (Stamina) & Mental (Wisdom) XP bars,
  six daily quests with progressive overload, hydration quick-add, run/plank
  timers, and the 66-day map.
- **Cards** — Leitner 5-box spaced repetition. Review due cards, earn Mental XP,
  add your own decks/cards. The Valmiki Ramayan deck is seeded on first launch.
- **Stats** — completion %, current/longest streak, XP split, weekly-wins chart,
  and unlockable achievements.
- **Settings** — edit/add/delete quests, book length, water goal, reminder time,
  `.ics` reminder export, and **backup export/import + reset**.

> Tip: iOS can clear web-app storage under pressure. Use **Settings → Export**
> regularly to keep a JSON backup.

## Reminders

iOS web apps can't schedule local notifications, so reminders are **local-first**:

- An in-app nudge on the Today screen when the day is incomplete.
- **Settings → Download Daily Reminder (.ics)** adds a recurring daily alert to
  your iOS Calendar that fires on the lock screen — no server needed.

## Optional lock-screen widget

iOS lock-screen widgets are native-only, so the PWA itself can't draw one. The
included [Scriptable](https://scriptable.app) script gives you a real
home/lock-screen widget that shows today's day number and quest targets,
computed from `data/schedule.json` (no backend):

1. Edit `widget/ramayan-widget.scriptable.js` and set `HOST` to your tinyhost URL.
2. In Scriptable, create a new script and paste the file.
3. Add a Scriptable widget to your screen and point it at that script.

The widget shows daily **targets** only; live check-marks/XP/streak stay in the app.

## Project structure

```
index.html              App shell + bottom tab nav
manifest.webmanifest    PWA manifest
sw.js                   Offline service worker (cache-first)
css/styles.css          Styles
js/
  app.js                Controller: routing, rendering, events
  state.js              Schema, persistence, migration, XP, backup
  quests.js             Daily targets, chronological engine, quest CRUD
  flashcards.js         Leitner SRS, deck/card CRUD
  stats.js              Metrics, achievements, inline SVG charts
  timers.js             Stopwatch/countdown + water counter
  reminders.js          Nudge text + .ics builder
  util.js               Date + DOM helpers
data/
  ramayan-deck.json     Seed flashcard deck
  schedule.json         Static config for the app + widget
icons/                  App icons (generated by tools/gen-icons.py)
widget/                 Optional Scriptable lock-screen widget
legacy/                 Original single-file version (backup)
```

## Regenerate icons

```bash
python3 tools/gen-icons.py
```

## Notes on this version

Rebuilt from the original single-file prototype into a modular PWA. Behavioural
fixes carried over: multi-level XP roll-over now uses a loop (no lost XP),
un-checking a quest correctly reverses level-ups, and the lopsided automatic
"passive XP" was removed so Physical and Mental progress fairly. The start date
remains hardcoded to 17 June 2026.
